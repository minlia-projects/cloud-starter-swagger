## 集成到自已的项目时添加依赖项    

```pom
<dependency>
  <groupId>com.minlia.cloud.starter</groupId>
  <artifactId>cloud-starter-swagger</artifactId>
  <version>2.0.0.RELEASE</version>
</dependency>
```
